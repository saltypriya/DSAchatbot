### Prerequisites

- Node.js v18+

### Installation

```bash
# Clone repository
git clone https://github.com/saltypriya/DSAchatbot.git
cd DSAchatbot

# Backend setup
cd backend
npm install
cp .env  # Add your Gemini API key

# Frontend setup
cd ../frontend
npm install

# Start both servers (in separate terminals)
# Backend:
cd ../backend && node server.js
# Frontend:
cd ../frontend && npm run dev
```
